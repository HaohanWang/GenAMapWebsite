Structured Association Mapping

GenAMap has standard association mapping methods, as well as cutting-edge methods that incorporate population and sample structure to improve accuracy. Current algorithms include:

  • Linear Regression (Lasso)
  • Multi-Population Lasso
  • Tree Lasso
  • Adaptive Multi-task Lasso
  • Graph-guided fused Lasso
