Structured Association Mapping

GenAMap offers cutting-edge machine learning methods for structured association mapping. Structured association mapping is the process of discovering genotype-phenotype assocations. Instead of examining a single SNP at a time, methods for structured association mapping consider all SNPs jointly, generating more accurate associations. Furthermore, these methods incorporate knowledge of sample and population structure to improve the accuracy and reliability of discovered associations. Current algorithms include:

  • Linear Regression (Lasso). This method assumes that there are few causal SNPs for each phenotype, and encourages sparsity in the resulting discovery. Details can be found in the peer-reviewed paper.
  • Multi-Population Lasso. This method assumes that different sub-populations may have differences in the sets of causal SNPs, but encourages similar sets within sub-populations. Details can be found in the peer-reviewed paper.
  • Tree Lasso. This method assumes a hierarchical structure to the response nodes, arranged in a tree structure. Details can be found in the peer-reviewed paper.
  • Adaptive Multi-task Lasso. This method uses similarities between traits to detect associations simultaneously for multiple traits. Details can be found in the peer-reviewed paper.
  • Graph-guided fused Lasso. This method assumes that the traits are arranged in a graph structure. Details can be found in the peer-reviewed paper.

Hypothesis Testing

Hypothesis testing is used to examine the potential association between a single nucleotide polymorphisms (SNPs) and a phentoype. GenAMap makes this process easy through intuitive an interface and visual results.

GenAMap current offers three standard hypothesis testing methods

  • Chi-Square Test
  • Fisher Exact Test
  • Wald Test

Confounding Effect Correction

When measuring genotypic and phenotypic data, confounding effects can make random noise look reliable. GenAMap offers methods to detect such confounding effects, and correct for it, including standard linear mixed model and sparse linear mixed model

  • Linear Mixed Model: standard model learning effect sizes while correcting confounding factors, based on famose FaST-LMM implementation proposed in this paper
  • Sparse Linear Mixed Model: is an extension of linear mixed model that can correct confounding factors while select SNPs that contribute to phenotype combinatorially. Implementation is based on this paper
